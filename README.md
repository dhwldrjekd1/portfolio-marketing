# 최동윤 — 광고·마케팅 AI 풀스택 포트폴리오

리뷰·고객 데이터를 마케팅 성과로 바꾸는 서비스를 만드는 개발자입니다.
프론트엔드부터 백엔드, AI 모델까지 직접 구현하고 실제 서비스로 연결합니다.

🔗 **Live:** https://dyy.kr
🛠 **Stack:** HTML · CSS · JavaScript (정적 사이트, 빌드 도구 없음)

## 주요 프로젝트

- **ReviewLens** — 리뷰 감성분석(ABSA)·하이브리드 추천·RAG 챗봇으로 광고 카피와 셀러 응답을 자동 생성하는 AI 서비스
  - Live: https://huggingface.co/spaces/dongyuns/reviewlens
  - Code: https://github.com/dhwldrjekd1/reviewlens-
- **FORME** — 회원 등급·할인·사이즈 추천·결제까지 구현한 풀스택 커머스 (3인 팀 · 팀장)
  - Code: https://github.com/dhwldrjekd1/FORME_shop_backend
- **Gentle Monster 스타일 쇼핑몰** — 구매검증 리뷰·3단계 결제·관리자 기능을 1인 풀스택으로 완성한 브랜드 커머스
  - Code: https://github.com/dhwldrjekd1/portfolio_shop_spring_boot
- **TAS** — YOLO11 차량 탐지 + PaddleOCR 번호판 인식 기반 교통 분석 시스템 (팀 프로젝트)
  - Code: https://github.com/angela860807/Traffic_Analytics_Proposal
